# Photential Website

This repository contains the source files for [Photential](https://photential.com), a B2B platform for capturing and submitting installation photos.

The site is a static HTML/CSS landing page hosted on GitHub Pages and connected to the custom domain photential.com.

## Deployment

- Hosted via GitHub Pages from the main branch root.
- Custom domain configured in GitHub Pages settings.
- DNS configured with A records pointing to GitHub Pages IPs and CNAME for www.

## How to update

- Edit `index.html` or other assets in the repo.
- Commit and push changes to trigger GitHub Pages rebuild.
